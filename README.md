# Metawork-fosis

Este repositorio contiene el código fuente para la página web de Metawork-fosis, diseñada para Baxto Style Tattoo.

## Descripción

La página web presenta un diseño interactivo con un fondo de partículas en movimiento y botones de redes sociales que orbitan alrededor de un avatar central. Incluye enlaces directos a WhatsApp, TikTok, Linktree y Google Maps.

## Estructura del Proyecto

- `public/index.html`: Archivo HTML principal con la estructura de la página.
- `public/css/style.css`: Hoja de estilos CSS para el diseño y las animaciones.
- `public/js/app.js`: Script JavaScript para la lógica de las partículas y la interactividad.
- `render.yaml`: Archivo de configuración para el despliegue en Render.com.

## Despliegue

Este proyecto está configurado para ser desplegado como un sitio estático en Render.com utilizando el archivo `render.yaml`.
